Panduan Singkat

1. Tahap Editing Aplikasi

Gunakan tools seperti APK Editor atau MT Manager untuk menyisipkan perintah terminal ke dalam `smali` atau `manifest` aplikasi target agar bisa di-hook oleh TrojaTools.

2. Setup di Termux

Jalankan perintah dasar berikut untuk mempersiapkan environment:

pkg update && pkg upgrade
pkg install python git
git clone https://github.com/username/trojatools.git
