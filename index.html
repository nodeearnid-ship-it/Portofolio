Firebase Integration

TrojaTools bekerja menggunakan perantara **Firebase Realtime Database**. Ini memungkinkan perintah dari terminal sampai ke HP secara instan melalui koneksi internet.

Langkah 1: APK

Pasang file google-services.json ke dalam APK yang dimodifikasi.

Langkah 2: Terminal

Konfigurasi API Key Firebase pada script Python di Termux/PC Anda.

PANDUAN INSTALASI

1. Edit APK

Pelajari teknik smali editing untuk menanamkan fungsi kontrol ke aplikasi target.

2. Termux Setup

Lakukan instalasi python, modul request, dan git clone project TrojaTools.

3. Hubungkan

Pastikan database online, jalankan script, dan mulai kendalikan perangkat.
