Mengapa TrojaTools Berbeda?

TrojaTools bukan sekadar script. Ini adalah metodologi untuk menghubungkan ekosistem Android dengan kekuatan CLI (Command Line Interface). Dengan arsitektur yang ringan, Anda tidak memerlukan server berbayar yang mahal.

Real-time Sync

Latency di bawah 1 detik menggunakan Firebase WebSocket.

Modifikasi Fleksibel

Bisa di-inject ke hampir semua aplikasi Android berbasis Java/Kotlin.

$ troja-cli status > Searching for active nodes... > Node [X-01] Connected (Online) $ troja-cli shell "input keyevent 26" > Executing: Toggle Power Button...

DATA ANDA, DATABASE ANDA.

Kami percaya pada kedaulatan data. TrojaTools dirancang agar **100% bergantung pada akun Firebase milik Anda pribadi**. Tidak ada server pusat, tidak ada pengumpulan data dari kami. Anda adalah pemilik infrastruktur kontrol Anda sendiri.

Daftar Perintah Populer

--get-sms

Membaca pesan masuk terakhir dari perangkat target secara jarak jauh.

--get-location

Mengambil koordinat GPS presisi dan menampilkannya di peta terminal.

--open-url [link]

Membuka link website tertentu di browser HP target secara otomatis.

--take-photo

Mengambil gambar melalui kamera depan/belakang dan mengirimnya ke DB.

--get-contacts

Menyalin daftar kontak telepon ke dalam format JSON di terminal Anda.

--toast [teks]

Memunculkan pesan pop-up singkat pada layar perangkat target.

FAQ

Apakah butuh akses Root?

Tergantung fitur. Untuk perintah dasar tidak memerlukan root, namun untuk akses sistem mendalam seperti kontrol layar penuh, akses root sangat disarankan.

Apakah aplikasi target bisa terdeteksi?

TrojaTools berjalan sebagai background service. Selama integrasi smali dilakukan dengan benar, aplikasi akan tetap terlihat normal bagi user biasa.
