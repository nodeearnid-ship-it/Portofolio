Firebase Backend

Proyek ini sepenuhnya terintegrasi dengan **Firebase Realtime Database**. Ini memungkinkan sinkronisasi perintah secara instan antara terminal Anda dan aplikasi yang sudah dimodifikasi di HP target.

Client Side

Masukkan google-services.json ke dalam APK menggunakan APK Editor/MT Manager untuk mengaktifkan sinkronisasi.

Console Side

Konfigurasi API Key di terminal untuk mengirim perintah (payload) ke database secara realtime.

Open Firebase Console

Cara Penginstalan

01

Edit Aplikasi

Ikuti tutorial PDF untuk menyisipkan script troja ke dalam aplikasi target agar bisa menerima perintah dari Firebase.

02

Setup Termux

Install python dan clone repository. Masukkan kredensial Firebase Anda ke dalam file config.

03

Execution

Jalankan terminal dan mulai kontrol HP Anda melalui perintah teks sederhana tanpa menyentuh layar.
